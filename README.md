# Kelihatannya Murah

Kuis edukasi finansial interaktif yang membantu kamu memahami biaya tersembunyi di balik cicilan dan pinjaman.

## Cara Main

1. Baca skenarionya
2. Tebak total uang yang keluar sampai lunas
3. Lihat reveal dan bandingkan dengan jawabanmu

## Fitur

- 9 skenario finansial realistis
- Penjelasan detail untuk setiap jawaban
- Perbandingan dengan harga bakso agar lebih mudah dipahami
- Skor dan refleksi di akhir permainan

## Teknologi

Dibuat dengan HTML, CSS, dan JavaScript vanilla. Tidak ada dependency eksternal.
